O projeto Learning_Git, é um aprendizado como o proprio nome já diz, nele desenvolvemos uma aplicação simples apenas para treinar comandos git.

Obs.: Os tipos de commit serão descritos abaixo

chore   - Inserção de codigo em algum ponto da aplicação, que não é feature e nem fix.
fix     - Resolução de algum bug
feature - Adição de alguma funcionalidade

Programas necessário
Yarn
Node.js
Git

Passo 1 - Checar se foi tudo devidamente instalado.
Os comandos abaixo devem ser executados no terminal
git --version 	(checa a versão do git)
node -v 	(checa a versão do node)
yarn -v		(checa a versão do yarn)

Obs.: Se der algum erro, ou não aparecer a versão, siga o recomendado na pagina da aplicação.

Passo 2 - Instale o express
Basta está dentro do diretorio e executar yarn add express.
